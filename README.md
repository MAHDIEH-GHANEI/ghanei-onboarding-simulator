# 🌐 Ghanei Intelligent System - Activation Simulator

An interactive web-based simulator demonstrating the user authentication process, desktop software activation, and hardware terminal registration via mobile (using a QR code).

---

## ✨ Key Features & Capabilities

* **Simultaneous Desktop & Mobile Simulation:** Live display of the desktop monitoring panel and the user's mobile screen in a single split-screen view, featuring automatic app unlocking upon mobile confirmation (Real-time Sync).
* **Offline QR Code Generation:** Local and 100% reliable rendering of QR codes without requiring internet access or external web service dependencies.
* **Multilingual Support (i18n):** Instant language switching between English (Default), Persian, and Arabic, complete with automatic layout orientation adjustments (RTL/LTR).
* **Smart Terminal Status Check:** Simulates hardware verification, seamlessly directing users to either a new terminal registration flow or direct dashboard login.
* **Continuous Support Access:** Integrated "Contact Support" touchpoints across every stage of the user journey.

---

## 💻 Device Compatibility & Execution

* **Zero Setup Required:** Built entirely on HTML5 and pure JavaScript (Vanilla JS), requiring no software installation, backend packages, or web servers.
* **Full Compatibility Across All OS & Browsers:**
  * **Desktop:** Fully operational on Windows, macOS, and Linux.
  * **Mobile & Tablet:** Fully compatible with Android and iOS.
  * **Browsers:** Optimized for Chrome, Firefox, Safari, and Edge.

### 🚀 How to Run:
Simply double-click the `ghanei-activation-simulator.html` (or `index.html`) file to open it directly in any modern web browser and test the activation workflow on the right-side mobile simulator!
